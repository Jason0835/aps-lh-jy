# 单控机台动态排产粒度规则

## Purpose

规范硫化排程中单控机台 `K1501L/R`、`K1502L/R` 的使用粒度，确保试制、量试、小批量 SKU 按单边机台独立排产，正规 SKU 按物理整机同步占用 L/R 两侧。

## Requirements

### Requirement: SKU 类型必须决定单控机台占用粒度

系统 MUST 按 SKU 类型判断单控机台占用粒度：

1. 试制 SKU、量试 SKU、小批量 SKU 使用单边粒度；
2. 小批量 SKU 的判断边界为 `数量 <= 小批量阈值`，默认阈值为 `100`；
3. 非试制、非量试、非小批量的正规 SKU 使用整机粒度；
4. 单边粒度 SKU 可独立使用 `K1501L`、`K1501R`、`K1502L`、`K1502R`；
5. 整机粒度 SKU 使用单控机台时，必须同步占用同一物理机台的 L/R 两侧。

#### Scenario: 数量等于 100 的 SKU 按小批量处理

- **WHEN** 正规施工阶段 SKU 的硫化余量等于小批量阈值 `100`
- **THEN** 系统 MUST 将该 SKU 标记为小批量
- **AND** 该 SKU 使用单控机台时 MUST 按单边粒度处理

### Requirement: 正规 SKU 单控候选必须成组生成

系统 MUST 在新增排产候选机台生成阶段对正规 SKU 执行整机候选判断：

1. L/R 两侧都通过既有硬性机台约束时，才允许生成单控整机候选；
2. 候选集合中同一物理单控机台最多保留一个代表候选，代表侧统一为左侧；
3. L/R 任一侧缺失、禁用、寸口不匹配、模具不满足或被其它 SKU 当前排程结果占用时，不得生成整机候选；
4. 正规 SKU 的单控候选参考时间 MUST 取 L/R 两侧释放时间的较晚值；
5. 普通机台存在时，正规 SKU 仍 MUST 优先普通机台，单控整机候选只能作为普通机台后的回落。

#### Scenario: 缺少配对侧时不得生成正规 SKU 单控候选

- **WHEN** 正规 SKU 只有 `K1501R` 通过候选机台硬过滤
- **AND** `K1501L` 不存在或未通过硬过滤
- **THEN** 系统 MUST NOT 生成 `K1501R` 单边候选

#### Scenario: L/R 收尾时间不同时按较晚释放时间上机

- **WHEN** 正规 SKU 使用 `K1501` 单控整机候选
- **AND** `K1501L` 释放时间早于 `K1501R`
- **THEN** 系统 MUST 以 `K1501R` 释放时间作为整机最早换模接续起点

### Requirement: 新增排产结果必须按粒度写入

系统 MUST 在新增排产结果写入阶段按粒度处理单控机台：

1. 单边粒度 SKU 只写入当前侧结果；
2. 整机粒度 SKU 必须同时写入 L/R 两条结果；
3. 整机粒度两侧结果的物料、生产开始时间、生产结束时间、班次计划量 MUST 一致；
4. 整机粒度扣减日计划和硫化余量时 MUST 按 L/R 两侧合计量消费；
5. 若整机扣账后只剩奇数尾量，系统 MUST 只落地可均分到两侧的偶数量，剩余尾量留给后续未排或滚动处理。

#### Scenario: 正规 SKU 写入两侧一致结果

- **WHEN** 正规 SKU 使用 `K1501` 单控整机排产成功
- **THEN** 系统 MUST 同时生成 `K1501L` 和 `K1501R` 两条排产结果
- **AND** 两条结果 MUST 具有相同物料、相同生产起止时间和相同班次计划量

### Requirement: 续作和换活字块不得生成正规 SKU 单边结果

系统 MUST 在续作、降模释放和换活字块链路中保持单控机台粒度一致：

1. 单边粒度 SKU 可独立续作、收尾、释放、换模和换活字块；
2. 正规 SKU 单控续作必须 L/R 两侧同物料同步续作；
3. 正规 SKU 若只有单侧续作，或配对侧物料不同，系统 MUST 阻断单边续作并记录未排原因；
4. 当前换活字块追加入口一次只生成单侧结果时，系统 MUST 阻断正规 SKU 在单控机台上追加单边换活字块；
5. 正规 SKU 单控释放和后续新增候选 MUST 以 L/R 两侧都释放为准。

#### Scenario: 正规 SKU 单侧续作被阻断

- **WHEN** `K1501L` 上存在正规 SKU 续作
- **AND** `K1501R` 不在同物料续作队列中
- **THEN** 系统 MUST NOT 生成 `K1501L` 单边正规续作结果
- **AND** 系统 MUST 记录“正规SKU单控机台L/R整机续作条件不满足，禁止单边续作”未排原因

### Requirement: 正规 SKU 单控降模必须 L/R 同步

系统 MUST 在续作降模减机台链路中对正规 SKU 单控整机执行 L/R 同步判断：

1. 降模保留机台选择时，正规 SKU 单控 L/R 两侧 MUST 同时保留或同时释放；
2. 不允许只保留 L 侧而释放 R 侧，或只保留 R 侧而释放 L 侧；
3. 正规 SKU 单控整机不进入单机降模链路，避免只保留一侧；
4. 试制、量试、小批量 SKU 降模仍按单边独立处理。

#### Scenario: 正规 SKU 降模时 L/R 同步保留

- **WHEN** 正规 SKU 在 `K1501L` 和 `K1501R` 上同时续作
- **AND** 降模判断只需保留一侧产能即可满足需求
- **THEN** 系统 MUST 同时保留 `K1501L` 和 `K1501R`，或同时释放两者
- **AND** 系统 MUST NOT 只保留 `K1501L` 而释放 `K1501R`

### Requirement: 正规 SKU 单控收尾释放必须 L/R 同步

系统 MUST 在续作收尾释放机台登记时对正规 SKU 单控整机执行 L/R 同步释放：

1. 正规 SKU 单控一侧释放时，系统 MUST 同步登记释放配对侧；
2. 不允许只释放一侧给其他 SKU 使用；
3. 试制、量试、小批量 SKU 收尾释放仍按单边独立处理。

#### Scenario: 正规 SKU 释放时配对侧同步释放

- **WHEN** 正规 SKU 在 `K1501L` 上续作结果被释放（降模或零计划移除）
- **THEN** 系统 MUST 同时登记释放 `K1501R`
- **AND** 后续正规 SKU 候选 MUST 等 `K1501L` 和 `K1501R` 都释放后才可使用

### Requirement: 单控机台反向匹配 SKU

系统 SHALL 在试制、量试、小批量 SKU 成功排上单控机台一侧后，尝试为配对侧反向查找可排 SKU：

1. 只有单边粒度 SKU（试制、量试、小批量）成功排上一侧后才触发反向匹配；
2. 反向匹配只查找试制、量试、小批量 SKU，正规 SKU 不参与；
3. 匹配优先级：试制 > 量试 > 小批量，同一层级内复用现有 SKU 排序规则；
4. 规格匹配顺序：同规格 > 同断面宽 > 同英寸；
5. 断面宽从规格中解析，复用 `ProductSpecificationsUtils`；
6. 匹配到的候选 SKU 仍需满足项目已有所有约束（硫化余量、月计划、模具、首检、班次等）；
7. 如果配对侧没有合适 SKU，允许空闲，不强制排产。

#### Scenario: 单边 SKU 排上一侧后为配对侧反向匹配

- **WHEN** 试制 SKU A 成功排上 `K1501L`
- **AND** `K1501R` 空闲
- **AND** 待排 SKU 中存在同规格的量试 SKU B
- **THEN** 系统 SHALL 记录 SKU B 为 `K1501R` 的反向匹配推荐
- **AND** SKU B 在后续选机时 SHALL 优先选择 `K1501R`

#### Scenario: 配对侧无匹配 SKU 时允许空闲

- **WHEN** 小批量 SKU 成功排上 `K1501L`
- **AND** `K1501R` 空闲
- **AND** 待排 SKU 中没有可匹配的试制、量试或小批量 SKU
- **THEN** 系统 SHALL 允许 `K1501R` 空闲
- **AND** 系统 MUST NOT 强制为 `K1501R` 排产

### Requirement: 保存前必须校验正规 SKU 单控整机一致性

系统 MUST 在排程结果保存前校验正规 SKU 单控结果：

1. 正规 SKU 在配置生效的单控 L/R 机台上有正排产量时，必须存在配对侧结果；
2. 配对侧结果必须是相同物料；
3. L/R 两侧生产开始时间、生产结束时间和各班次计划量必须一致；
4. 校验失败时 MUST 抛出排程结果校验异常，禁止结果落库；
5. 试制、量试、小批量 SKU 不适用该整机一致性校验。

#### Scenario: 单边正规结果保存前被拦截

- **WHEN** 排程结果中存在正规 SKU `K1501L` 正排产量
- **AND** 不存在同物料 `K1501R` 配对结果
- **THEN** 系统 MUST 在保存前抛出结果校验异常
