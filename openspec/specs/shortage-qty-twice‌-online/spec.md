## Purpose

防止仅因本月历史欠产进入新增排产、且后续已经没有月计划的 SKU 在最近一次已有实际完成量后再次上机，避免重复补排。

## Requirements

### Requirement: 仅历史欠产 SKU 按最近一次完成量判断是否跳过

系统 MUST 在 S4.5 新增排产前判断 SKU 是否同时满足：

- 本月 T 日前累计历史欠产量大于 0；
- T～月底月计划 dayN 均为 0；
- SKU 不是续作补偿 SKU；
- 当前排程月份月初至 T-1 最近一次非空日完成量大于 0。

全部满足时，系统 MUST 不再新增排产，并将 SKU 写入未排结果。

#### Scenario: T-1 无数据但更早最近一次完成量大于0

- **GIVEN** SKU 仅存在本月历史欠产且 T～月底无月计划
- **AND** T-1 没有非空日完成量
- **AND** 当前月更早的最近一次非空日完成量大于 0
- **WHEN** 系统执行新增排产
- **THEN** 系统不生成该 SKU 的新增排程结果
- **AND** 系统将该 SKU 写入未排结果

#### Scenario: 最近一次非空完成量为0

- **GIVEN** SKU 仅存在本月历史欠产且 T～月底无月计划
- **AND** 当前月最近一次非空日完成量为 0
- **AND** 更早日期存在大于 0 的完成量
- **WHEN** 系统执行新增排产
- **THEN** 系统以最近一次的 0 为准，不继续向更早日期回溯
- **AND** 系统继续执行原有历史欠产补排判断

#### Scenario: 完成量为空时继续回溯

- **GIVEN** 当前月较近日期的 DAY_FINISH_QTY 为 NULL
- **AND** 更早日期存在非空完成量
- **WHEN** 系统解析最近一次完成量
- **THEN** 系统忽略 NULL，并继续向前查找最近一次非空完成量

#### Scenario: 按产品状态读取逐日完成量

- **GIVEN** 基础数据按“物料 + 产品状态 + 完成日期”写入当前月逐日完成量
- **AND** SKU 存在产品状态
- **WHEN** 系统解析最近一次完成量
- **THEN** 系统 MUST 优先使用 SKU 产品状态匹配逐日完成量
- **AND** 不得因只按物料编码查找而漏判已完成记录

#### Scenario: 不跨月读取完成量

- **GIVEN** T 为当前月第一日
- **AND** 上月最后一日存在大于 0 的完成量
- **WHEN** 系统解析最近一次完成量
- **THEN** 系统不得使用上月完成量触发本规则

#### Scenario: 跨月窗口下月远期有计划不阻断当前月跳过

- **GIVEN** T 为当前月月末且排程窗口跨到下月
- **AND** SKU 在 T～窗口结束日的月计划 dayN 均为 0
- **AND** SKU 在下月窗口后存在远期月计划
- **AND** SKU 当前月月初至 T-1 最近一次非空日完成量大于 0
- **WHEN** 系统执行新增排产前置判断
- **THEN** 系统不得因下月窗口后的远期月计划阻断本规则
- **AND** 系统不生成该 SKU 的新增排程结果
- **AND** 系统将该 SKU 写入未排结果

#### Scenario: 胎胚库存收尾 SKU 仍适用本规则

- **GIVEN** SKU 命中成型胎胚库存收尾
- **AND** SKU 同时满足仅历史欠产、T～月底无月计划、最近一次非空日完成量大于 0
- **WHEN** 系统执行新增排产前置判断
- **THEN** 系统 MUST 优先按本规则跳过新增排产
- **AND** 系统不得因胎胚库存收尾状态绕过本规则

### Requirement: 未排原因与日志

命中跳过规则时，系统 MUST 使用未排原因“仅历史欠产、后续无月计划，且最近一次（前一次）已有完成量，本次跳过不排”，并输出包含 SKU、欠产量、排程日期、最近一次日完成量和跳过原因的日志。

#### Scenario: 命中规则后可对账

- **WHEN** SKU 命中最近一次完成量跳过规则
- **THEN** 未排结果数量等于本次本月历史欠产量
- **AND** 未排原因等于固定文案
- **AND** 日志包含 SKU、欠产量、排程日期、最近一次日完成量和跳过原因

### Requirement: 规则影响边界

本规则 MUST 仅作用于新增排产前置判断，不得改变续作、续作补偿、换模、换活字块、胎胚库存、日计划扣减和机台匹配逻辑。

#### Scenario: 续作补偿 SKU 不受影响

- **GIVEN** SKU 为续作补偿 SKU
- **WHEN** 系统执行新增排产承接
- **THEN** 系统不得因本规则跳过该 SKU
- **AND** 系统继续执行原续作补偿逻辑
