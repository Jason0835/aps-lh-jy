# 续作排产降模减机台规则

## Purpose

统一硫化排程续作多机台降模减机台规则，确保释放的只是超过 dayN 节奏、收尾目标和既有资源约束所需的冗余机台，同时避免首日无计划但后续仍有计划的 MES 在机 SKU 被错误转为新增换模。

## Requirements

### Requirement: 续作降模必须受 dayN 最小机台数保护

系统必须（MUST）按业务日 dayN 日计划量和 SKU 日标准产量计算最小续作机台数，并将该机台数作为多台续作降模硬下限。

SKU 日标准产量优先取硫化日标准产量主数据；无主数据时取 SKU 日产能；仍无有效值时取 `班产 * 3`。

#### Scenario: 恒定 dayN 不得低于最小续作机台数

- **WHEN** 同一 SKU 存在多台 MES 在机续作机台
- **AND** 月计划日计划为 96、96、96
- **AND** SKU 日标准产量为 48
- **THEN** dayN 最小续作机台数为 2
- **AND** 系统必须保留 2 台续作机台
- **AND** 不得因首日完成量扣减或后续追补需求为 0 释放其中 1 台

#### Scenario: dayN 下降时允许逐日释放冗余机台

- **WHEN** 同一 SKU 月计划 dayN 从前两日高计划下降到后续低计划
- **AND** 后续业务日所需最小续作机台数低于当前续作机台数
- **THEN** 系统可以释放超过后续 dayN 最小机台数的冗余机台
- **AND** 释放后仍必须满足收尾目标、晚班不可换模、清洗、保养和既有资源约束

### Requirement: 首日无计划但后续有计划的续作必须保留续作身份

系统必须（MUST）在 MES 在机同物料 SKU 首日 dayN 为 0 但后续窗口存在正日计划时，保留续作身份并从后续首个正日计划班次继续续作。

#### Scenario: 首日无计划但后续有计划时保留续作身份

- **WHEN** MES 在机同物料 SKU 首日 dayN 为 0
- **AND** 后续窗口存在正日计划
- **THEN** 系统必须继续按续作排产承接该 SKU
- **AND** 起排时间推进到后续首个正日计划班次
- **AND** 不得生成同物料新增换模记录

## 规则一：适用范围

续作排产中，只有同一 SKU 当前存在多台续作机台时，才需要判断是否降模减机台。

单机台续作、新增排产、换活字块排产不适用该规则。

---

## 规则二：降模判断口径

系统应按 T～T+2 三天、8 个班次重新评估该 SKU 的最小可行续作机台数。

若减少部分机台后，剩余机台仍能满足以下要求，则允许减机台：

* 当前日月计划量；
* 本月欠产追补；
* SKU 硫化余量；
* SKU 收尾目标量；
* 试制 SKU 严格按日计划排产；
* 晚班不可换模限制；
* 非收尾续作排满可用产能规则；
* 收尾 SKU 不允许超排规则。

减机台数量：

```text
减机台数量 = 当前续作机台数 - 最小可行续作机台数
```

若结果小于等于 0，则不减机台。


## 补充规则

续作排产降模减机台时，系统应以 T 日当前日计划量作为减机台下限约束。

当尝试减少机台后，保留机台数已经是满足 T 日当前日计划量的最小机台数时，系统不得继续减少机台。

判断口径为：
保留 N 台机台可以满足 T 日当前日计划量；
但保留 N-1 台机台无法满足 T 日当前日计划量；
则 N 台为 T 日所需最小机台数，本轮降模判断应停止继续向下减少。
需逐日（T~T+2）按照以上规则判断是否减机台

但最终保留机台数仍需同时满足 T～T+2 窗口内的后续日计划、SKU 收尾目标量、硫化余量、晚班不可换模、试制严格日计划等约束。

当续作 SKU 的实际目标量大于 T～T+2 窗口日计划量汇总时，系统仍必须先按 dayN 判断是否存在冗余续作机台，不得因为“目标量大于窗口日计划”直接跳过降模减机台。

处理口径为：

* dayN 用于判断当前续作机台数是否超过当前日和后看日计划节奏需要；
* SKU 目标量、硫化余量、欠产追补和晚班不可换模规则用于降模后的保留机台重分配；
* 减机台后 SKU 总目标量不能被 dayN 改小；
* 若保留机台数已能满足当前日和后看日计划节奏，多余续作机台必须释放给后续换活字块或换模新增链路。
* 释放机台后若仍需回流新增链路补偿，补偿 SKU 的目标量必须按业务目标缺口计算，dayN 只能判断是否需要增机台，不得把非收尾补偿量压成日计划小缺口。

dayN 最小续作机台数为硬下限：

```text
dayN最小续作机台数 = CEIL(dayN日计划量 / SKU日标准产量)
```

SKU 日标准产量优先取硫化日标准产量主数据；无主数据时取 SKU 日产能；仍无有效值时取 `班产 * 3`。多台续作降模只能释放超过当前日和后看日 dayN 最小续作机台数的机台，不得因排程日完成量扣减、目标剩余量较小、硫化余量剩余或后续追补需求为 0，将保留机台数降到 dayN 最小续作机台数以下。

首日无计划但后续窗口有正日计划的 MES 在机同物料 SKU，必须保留续作身份，不得释放原机台后转为新增换模。该 SKU 的续作起排时间应推进到后续首个正日计划班次；窗口完全无计划且无历史欠产时，仍沿用既有无计划释放规则。

降模释放标记只能在机台实际从续作结果中释放后登记；未释放的续作机台不得写入释放集合，也不得阻断后续补偿判断。

### Scenario: 目标量大于窗口日计划时仍按 dayN 释放冗余机台

- **WHEN** 同一 SKU 存在多台续作机台，且 SKU 实际目标量大于 T～T+2 窗口日计划量汇总
- **AND** 当前日或后看日计划节奏只需要更少机台即可满足
- **THEN** 系统必须进入按业务日降模减机台判断
- **AND** 保留机台数必须按 dayN 节奏计算
- **AND** 释放的续作机台不得继续占用后续可换模新增产能
- **AND** SKU 总目标量继续由降模后的保留机台按续作规则重分配，不得被 dayN 直接裁小

### Scenario: 续作补偿不得被 dayN 剩余额度压小

- **WHEN** 非收尾续作 SKU 已释放冗余机台，且原续作排产后仍存在业务目标缺口
- **AND** dayN 节奏判断确认仍需要进入新增链路补偿机台产能
- **THEN** 续作补偿量必须按业务目标缺口进入新增排产
- **AND** dayN 只继续参与新增链路的加机台节奏模拟
- **AND** 系统不得用 dayN 剩余额度或 day1 + day2 + day3 窗口计划汇总压小非收尾补偿量

### Scenario: 恒定 dayN 不得低于最小续作机台数

- **WHEN** 同一 SKU 存在多台 MES 在机续作机台
- **AND** 月计划日计划为 96、96、96
- **AND** SKU 日标准产量为 48
- **THEN** dayN 最小续作机台数为 2
- **AND** 系统必须保留 2 台续作机台
- **AND** 不得因首日完成量扣减或后续追补需求为 0 释放其中 1 台

### Scenario: 首日无计划但后续有计划时保留续作身份

- **WHEN** MES 在机同物料 SKU 首日 dayN 为 0
- **AND** 后续窗口存在正日计划
- **THEN** 系统必须继续按续作排产承接该 SKU
- **AND** 起排时间推进到后续首个正日计划班次
- **AND** 不得生成同物料新增换模记录

## 规则三：减哪台机台

降模减机台时，应优先保留更适合继续生产的机台。

保留机台优先级：

1. 胶囊已使用次数多的机台优先保留；
2. 胶囊次数相同，机台编码小的优先保留。

释放机台优先级：

1. 胶囊已使用次数少的机台优先释放；
2. 胶囊次数相同，机台编码大的优先释放。

---

## 规则四：减机台后重新分配产量

减机台后，SKU 总目标量不能改变。

系统应将该 SKU 后续排产量重新分配到保留的续作机台上。

非收尾 SKU：

* 满足日计划和欠产追补后，可按续作规则补满可用产能；
* 晚班不可换模时，应尽量避免晚班产能浪费。

收尾 SKU：

* 严格按收尾目标量排产；
* 不允许因补满班产、晚班不可换模或减机台重分配而超排；
* 最后一个班次允许不足班产。

---

## 规则五：收尾目标量

收尾 SKU 的目标量按胎胚是否共用区分。

单胎胚 / 非共用胎胚：

```text
收尾目标量 = MAX(硫化余量, 胎胚库存)
```

共用胎胚：

```text
收尾目标量 = 硫化余量
```

若共用胎胚 SKU 的硫化余量为 0，则不排产，并列入未排结果。

---

## 规则六：晚班不可换模处理

若机台在中班结束时准备下机，且下一班为晚班，由于晚班不可换模，需特殊处理。

非收尾 SKU：

* 若释放机台后其他 SKU 无法在晚班换模上机，则当前 SKU 可继续排满下一个晚班。

收尾 SKU：

* 仍然必须严格控制收尾目标量；
* 不得为了填满晚班而超排。

---

## 规则七：机台收尾不等于 SKU 收尾

同一 SKU 多机台续作时，单台机台完成任务只表示该机台可以释放，不代表 SKU 已收尾。

SKU 是否收尾必须按 SKU 维度汇总所有机台的累计排产量判断。

```text
SKU 累计排产量 >= SKU 收尾目标量
```

才表示 SKU 收尾完成。

---

## 规则八：结构收尾不参与降模直接判断

结构五天内收尾只用于 SKU 排序判断，不作为续作降模减机台的直接条件。

续作降模减机台应基于 SKU 当前续作机台数、窗口需求、日计划量、欠产、余量、收尾状态和机台可用性判断。

不得混用：

* 结构收尾；
* SKU 收尾；
* 机台收尾。
