# 续作排产降模减机台规则

## Purpose

统一硫化排程续作多机台降模减机台规则，确保释放的只是超过 dayN 节奏、收尾目标和既有资源约束所需的冗余机台，同时避免首日无计划但后续仍有计划的 MES 在机 SKU 被错误转为新增换模。

## Requirements

### Requirement: T 日计划量与 T-1 日相等时不得降模

系统必须（MUST）在续作 SKU 的 T 日降模计算入口比较 T 日与 T-1 日原始月计划日计划量。若两者相等，T 日必须维持 MES 在线续作机台数，不得执行降模减机台。

该入口保护必须先于收尾单机可覆盖、首日收口等快捷降模分支生效。非收尾续作仍沿用既有满产规则，受保护机台应按清洗、停机、保养等损失扣减后的各班次有效产能排满；严格收尾场景不得突破目标量，当顺序分配会使后序在线机台计划量为 0 时，应在全部受保护机台间分配当日需求，避免后置零计划收口将受保护机台释放。

该保护仅对 T 日当前业务日生效，不得写入 SKU 或排程上下文形成整个窗口的禁止降模状态。T+1、T+2 等后续日期必须重新执行既有最小机台数、后看欠产、收尾目标、晚班不可换模和资源约束判断，满足条件时仍可正常减少机台。

比较值必须来自定稿月计划 `DAY_1～DAY_31` 原始日计划量，不得使用实际排产量、日计划剩余额度、欠产合入量或班次计划量。T 日为月初时，必须分别按所属年月读取上月最后一日和本月第一日月计划。

#### Scenario: T 日计划量与 T-1 日相等时保留全部续作机台

- **WHEN** 同一续作 SKU 存在 2 台 MES 在线续作机台
- **AND** T-1 日原始月计划量为 48
- **AND** T 日原始月计划量为 48
- **THEN** T 日必须保留 2 台续作机台
- **AND** 不得执行 T 日降模释放

#### Scenario: T 日跳过降模不影响后续日期重新判断

- **WHEN** 续作 SKU 因 T 日与 T-1 日原始月计划量相等而跳过 T 日降模
- **AND** T+1 日按既有规则只需更少续作机台
- **THEN** T+1 日必须重新执行降模判断
- **AND** 满足既有降模条件时可以正常释放冗余机台

#### Scenario: 月初跨月比较原始日计划量

- **WHEN** T 日为本月 1 日
- **THEN** 系统必须读取上月最后一日作为 T-1 日原始月计划量
- **AND** 读取本月 1 日作为 T 日原始月计划量
- **AND** 不得将两个日期映射到同一条月计划记录

### Requirement: 续作降模必须受 dayN 最小机台数保护

系统必须（MUST）按业务日 dayN 日计划量和 SKU 日标准产量计算最小续作机台数，并将该机台数作为多台续作降模硬下限。

对于非严格收尾续作，该计算结果同时是当前业务日和后看日的最终保留机台数决策。系统不得再使用`班产 × 可用班次数`、部分清洗、部分停机或部分保养后的实际产能，将保留机台数反向扩大。保留机台实际排量不足 dayN 时，差额必须进入既有滚动差额和日计划欠产账本；已经按日标准量判定释放的机台，除完成必要的不可换模边界晚班外，不得参与当日补量，也不得被同物料续作补偿或新增链路重新补回。

SKU 日标准产量优先取硫化日标准产量主数据；无主数据时取 SKU 日产能；仍无有效值时取 `班产 * 3`。

#### Scenario: 恒定 dayN 不得低于最小续作机台数

- **WHEN** 同一 SKU 存在多台 MES 在机续作机台
- **AND** 月计划日计划为 96、96、96
- **AND** SKU 日标准产量为 48
- **THEN** dayN 最小续作机台数为 2
- **AND** 系统必须保留 2 台续作机台
- **AND** 不得因首日完成量扣减或后续追补需求为 0 释放其中 1 台

#### Scenario: dayN 下降时允许逐日释放冗余机台

- **WHEN** 同一 SKU 月计划 dayN 从前两日高计划下降到后续低计划
- **AND** 后续业务日所需最小续作机台数低于当前续作机台数
- **THEN** 系统可以释放超过后续 dayN 最小机台数的冗余机台
- **AND** 释放后仍必须满足收尾目标、晚班不可换模、清洗、保养和既有资源约束

#### Scenario: 日标准量决定机台数且实际产能不足进入欠产

- **WHEN** 同一非收尾续作 SKU 存在 3 台 MES 在机机台
- **AND** 原始 dayN 为 `150,100,100`
- **AND** SKU 硫化日标准产量为 50、班产为 16
- **AND** T 日与 T-1 日原始计划量均为 150
- **THEN** T 日必须保留 3 台续作机台
- **AND** T+1、T+2 所需续作机台数必须为 `CEIL(100 / 50) = 2`
- **AND** 两台机台按实际可用班产排产后形成的不足必须进入欠产账本
- **AND** 系统不得因为两台物理日产能只有 96，或其中一台存在部分清洗，再保留或补回第三台
- **AND** 第三台完成必要的不可换模边界晚班后，后续早班、中班必须清零并真实释放

### Requirement: 首日无计划但后续有计划的续作必须保留续作身份

系统必须（MUST）在 MES 在机同物料 SKU 首日 dayN 为 0 但后续窗口存在正日计划时，保留续作身份并从后续首个正日计划班次继续续作。

#### Scenario: 首日无计划但后续有计划时保留续作身份

- **WHEN** MES 在机同物料 SKU 首日 dayN 为 0
- **AND** 后续窗口存在正日计划
- **THEN** 系统必须继续按续作排产承接该 SKU
- **AND** 起排时间推进到后续首个正日计划班次
- **AND** 不得生成同物料新增换模记录

### Requirement: 正规 SKU 单控续作必须 L/R 整机同步

系统必须（MUST）在续作排产、降模减机台和释放登记中按单控动态粒度处理：

1. 试制、量试、小批量 SKU 在单控机台上可以按 L/R 单边独立续作、收尾和释放；
2. 正规 SKU 在单控机台上必须 L/R 两侧同物料同步续作；
3. 若正规 SKU 只有单侧续作，或配对侧不是同物料正规 SKU，系统不得生成单边续作结果；
4. 正规 SKU 单控续作释放给新增链路时，后续正规 SKU 候选必须等 L/R 两侧都释放；
5. 正规 SKU 单控续作不满足整机条件时，必须记录明确未排原因，不得通过兜底补齐或吞异常继续排产。

#### Scenario: 正规 SKU 单侧续作不得生成结果

- **WHEN** `K1501L` 为正规 SKU 续作机台
- **AND** `K1501R` 不在同物料续作队列中
- **THEN** 系统必须阻断 `K1501L` 单边续作结果
- **AND** 系统必须记录正规 SKU 单控整机续作条件不满足的未排原因

## 规则一：适用范围

续作排产中，只有同一 SKU 当前存在多台续作机台时，才需要判断是否降模减机台。

单机台续作、新增排产、换活字块排产不适用该规则。

---

## 规则二：降模判断口径

系统应按 T～T+2 三天、8 个班次重新评估该 SKU 的续作机台数。非严格收尾续作先按当前业务日和后看日原始 dayN 分别计算机台数，并取观察窗口内最大值作为最终保留数量；严格收尾继续受目标量上限和既有资源约束控制。

若减少部分机台后，剩余机台仍能满足以下要求，则允许减机台：

* 当前日及后看日原始月计划量；
* 本月欠产追补；
* SKU 硫化余量；
* SKU 收尾目标量；
* 试制 SKU 严格按日计划排产；
* 晚班不可换模限制；
* 非收尾续作保留机台排满可用产能规则；
* 收尾 SKU 不允许超排规则。

减机台数量：

```text
减机台数量 = 当前续作机台数 - 最小可行续作机台数
```

若结果小于等于 0，则不减机台。


## 补充规则

续作排产降模减机台时，系统应以 T 日当前日计划量作为减机台下限约束。

当尝试减少机台后，保留机台数已经是满足 T 日当前日计划量的最小机台数时，系统不得继续减少机台。

判断口径为：
保留 N 台机台可以满足 T 日当前日计划量；
但保留 N-1 台机台无法满足 T 日当前日计划量；
则 N 台为 T 日所需最小机台数，本轮降模判断应停止继续向下减少。
需逐日（T~T+2）按照以上规则判断是否减机台

非严格收尾续作的最终保留机台数以当前日和后看日 dayN 最小续作机台数的最大值为准，不得再按实际班产扩大。SKU 收尾目标量、硫化余量、晚班不可换模、试制严格日计划等约束继续控制实际排量、释放时点和严格目标上限，不得改变该机台数口径。

当续作 SKU 的实际目标量大于 T～T+2 窗口日计划量汇总时，系统仍必须先按 dayN 判断是否存在冗余续作机台，不得因为“目标量大于窗口日计划”直接跳过降模减机台。

处理口径为：

* dayN 用于判断当前续作机台数是否超过当前日和后看日计划节奏需要；
* SKU 目标量、硫化余量、欠产追补和晚班不可换模规则用于降模后的保留机台重分配；
* 减机台后 SKU 总目标量不能被 dayN 改小；
* 若保留机台数已能满足当前日和后看日计划节奏，多余续作机台必须释放给后续换活字块或换模新增链路。
* 释放机台后若仍需回流新增链路补偿，补偿 SKU 的目标量必须按业务目标缺口计算，dayN 只能判断是否需要增机台，不得把非收尾补偿量压成日计划小缺口。

dayN 最小续作机台数为硬下限：

```text
dayN最小续作机台数 = CEIL(dayN日计划量 / SKU日标准产量)
```

SKU 日标准产量优先取硫化日标准产量主数据；无主数据时取 SKU 日产能；仍无有效值时取 `班产 * 3`。多台续作降模只能释放超过当前日和后看日 dayN 最小续作机台数的机台，不得因排程日完成量扣减、目标剩余量较小、硫化余量剩余或后续追补需求为 0，将保留机台数降到 dayN 最小续作机台数以下。

对于非严格收尾续作，当前日和后看日的 dayN 最小续作机台数不仅是下限，也是最终机台数决策。若日标准产量大于`班产 × 3`，或保留机台存在部分清洗、部分停机、部分保养，系统仍必须按日标准产量计算机台数；实际少排量进入既有欠产账本，不得调用当日补量选机逻辑重新使用已释放机台。整日完全不可用机台、严格收尾和正规 SKU 单控 L/R 整组约束继续沿用现有硬约束。

首日无计划但后续窗口有正日计划的 MES 在机同物料 SKU，必须保留续作身份，不得释放原机台后转为新增换模。该 SKU 的续作起排时间应推进到后续首个正日计划班次；窗口完全无计划且无历史欠产时，仍沿用既有无计划释放规则。

降模释放标记只能在机台实际从续作结果中释放后登记；未释放的续作机台不得写入释放集合，也不得阻断后续补偿判断。

### Scenario: 目标量大于窗口日计划时仍按 dayN 释放冗余机台

- **WHEN** 同一 SKU 存在多台续作机台，且 SKU 实际目标量大于 T～T+2 窗口日计划量汇总
- **AND** 当前日或后看日计划节奏只需要更少机台即可满足
- **THEN** 系统必须进入按业务日降模减机台判断
- **AND** 保留机台数必须按 dayN 节奏计算
- **AND** 释放的续作机台不得继续占用后续可换模新增产能
- **AND** SKU 总目标量继续由降模后的保留机台按续作规则重分配，不得被 dayN 直接裁小

### Scenario: 续作补偿不得被 dayN 剩余额度压小

- **WHEN** 非收尾续作 SKU 已释放冗余机台，且原续作排产后仍存在业务目标缺口
- **AND** dayN 节奏判断确认仍需要进入新增链路补偿机台产能
- **THEN** 续作补偿量必须按业务目标缺口进入新增排产
- **AND** dayN 只继续参与新增链路的加机台节奏模拟
- **AND** 系统不得用 dayN 剩余额度或 day1 + day2 + day3 窗口计划汇总压小非收尾补偿量

### Scenario: 恒定 dayN 不得低于最小续作机台数

- **WHEN** 同一 SKU 存在多台 MES 在机续作机台
- **AND** 月计划日计划为 96、96、96
- **AND** SKU 日标准产量为 48
- **THEN** dayN 最小续作机台数为 2
- **AND** 系统必须保留 2 台续作机台
- **AND** 不得因首日完成量扣减或后续追补需求为 0 释放其中 1 台

### Scenario: 首日无计划但后续有计划时保留续作身份

- **WHEN** MES 在机同物料 SKU 首日 dayN 为 0
- **AND** 后续窗口存在正日计划
- **THEN** 系统必须继续按续作排产承接该 SKU
- **AND** 起排时间推进到后续首个正日计划班次
- **AND** 不得生成同物料新增换模记录

## 规则三：减哪台机台

降模减机台时，应优先保留更适合继续生产的机台。

保留机台优先级：

1. 在机模具共用性数量少的机台优先保留；
2. 模具共用性相同，胶囊最大使用次数多的机台优先保留；
3. 模具共用性和胶囊最大使用次数都相同，机台编码小的优先保留。

释放机台优先级：

1. 在机模具共用性数量多的机台优先释放；
2. 模具共用性相同，胶囊最大使用次数少的机台优先释放；
3. 模具共用性和胶囊最大使用次数都相同，机台编码大的优先释放。

模具共用性取硫化在机信息中的在机模具号，多个模具号按英文逗号拆分后，匹配 SKU 与模具关系数据，按项目现有口径统计关联 SKU 数；数量越大，表示该机台在机模具可共用的 SKU 越多，越应优先释放。

胶囊最大使用次数取 `LhRepairCapsule` 中 `MAX(replaceCapsuleCount, replaceCapsuleCount2)`；若机台无胶囊使用记录，则按 0 参与排序。

排序方向：

```text
释放排序 = 模具共用性数量降序 -> 胶囊最大使用次数升序 -> 机台编码降序
保留排序 = 模具共用性数量升序 -> 胶囊最大使用次数降序 -> 机台编码升序
```

## 规则四：减机台后重新分配产量

减机台后，SKU 总目标量不能改变。

系统应将该 SKU 后续排产量重新分配到保留的续作机台上。

非收尾 SKU：

* 满足日计划和欠产追补后，可按续作规则补满可用产能；
* 晚班不可换模时，应尽量避免晚班产能浪费。

收尾 SKU：

* 严格按收尾目标量排产；
* 不允许因补满班产、晚班不可换模或减机台重分配而超排；
* 最后一个班次允许不足班产。

---

## 规则五：收尾目标量

收尾 SKU 的目标量按胎胚是否共用区分。

单胎胚 / 非共用胎胚：

```text
收尾目标量 = MAX(硫化余量, 胎胚库存)
```

共用胎胚：

```text
收尾目标量 = 硫化余量
```

若共用胎胚 SKU 的硫化余量为 0，则不排产，并列入未排结果。

---

## 规则六：晚班不可换模处理

若机台在中班结束时准备下机，且下一班为晚班，由于晚班不可换模，需特殊处理。

非收尾 SKU：

* 若释放机台后其他 SKU 无法在晚班换模上机，且当前 SKU 硫化余量充足，则当前 SKU 可继续排满下一个晚班。
* 若当前 SKU 剩余硫化余量不足以补满下一个晚班，则只允许按实际剩余可排量排产，不得强行超排。
* 即使保留机台已满足当前 dayN 需求、待下机机台当日剩余需求为 0，只要该待下机机台的释放点后紧接不可换模晚班，仍必须先按硫化余量补满该晚班，再释放机台。
* 待下机机台补满晚班后，排程结果的班次计划量、班次结束时间、`SPEC_END_TIME`、`T_DAY_SPEC_END_TIME` 以及运行态机台的 `estimatedEndTime` 必须同步更新到实际晚班结束时间。

收尾 SKU：

* 仍然必须严格控制收尾目标量；
* 不得为了填满晚班而超排。

#### Scenario: 当日剩余需求为 0 的降模下机机台仍需补满不可换模晚班

- **WHEN** 同一非收尾 SKU 存在多台续作机台
- **AND** dayN 下降后保留机台已满足当前日计划需求
- **AND** 待下机机台最后有量班次为中班，下一班为不可换模晚班
- **AND** 当前 SKU 硫化余量足以覆盖下一晚班
- **THEN** 系统必须先将待下机机台补满下一晚班
- **AND** 晚班结束后再释放该机台
- **AND** 该机台排程结果和运行态机台预计收尾时间必须同步到晚班结束时间

---

## 规则七：机台收尾不等于 SKU 收尾

同一 SKU 多机台续作时，单台机台完成任务只表示该机台可以释放，不代表 SKU 已收尾。

SKU 是否收尾必须按 SKU 维度汇总所有机台的累计排产量判断。

```text
SKU 累计排产量 >= SKU 收尾目标量
```

才表示 SKU 收尾完成。

---

## 规则八：结构收尾不参与降模直接判断

结构五天内收尾只用于 SKU 排序判断，不作为续作降模减机台的直接条件。

续作降模减机台应基于 SKU 当前续作机台数、窗口需求、日计划量、欠产、余量、收尾状态和机台可用性判断。

不得混用：

* 结构收尾；
* SKU 收尾；
* 机台收尾。

---

## 规则九：同物料多产品状态续作身份

续作规则中的“同一 SKU”必须（MUST）指物料编码与产品状态同时一致。

MES 或滚动衔接的在机产品状态为空时，必须按正规 `S` 解析续作身份；非空状态必须先完成全部精确匹配，未知状态才可按既有物料降级规则承接剩余 SKU。

同物料、同产品状态的多台在机机台必须共享该状态的日计划、目标量与实际消费账本；不同产品状态不得共享上述 SKU 账本。

#### Scenario: 两台空状态在机机台承接正规SKU

- **WHEN** 同物料同时存在 `S`、`T`、`X` 月计划
- **AND** 两台 MES 在机机台的产品状态均为空
- **THEN** 两台机台都必须承接 `S` 续作
- **AND** 两台机台共享 `S` 的同一份日计划和目标量账本
- **AND** `T`、`X` 保留独立的待排资格

#### Scenario: 精确产品状态优先于物料降级

- **WHEN** 未知产品状态在机记录排在 `T` 状态在机记录之前
- **AND** 月计划同时存在 `S` 与 `T`
- **THEN** 系统必须先将 `T` 在机记录精确匹配到 `T` SKU
- **AND** 未知状态只能在精确候选保留后对剩余 SKU 执行物料降级
