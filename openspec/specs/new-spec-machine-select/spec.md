## Purpose

规范新增排产阶段 SKU 选择候选硫化机台时的排序规则，确保系统先锁定最早参考收尾时间所在班次内的机台池，再在该池内按单控拆分、胎胚、模壳、规格、胶囊、英寸和机台编码逐层选择最优机台。

## Requirements

### Requirement: 新增排产选机台必须先按最早参考收尾时间所在班次筛同班次窗口

系统 MUST 在新增排产 SKU 选择候选机台时，先对通过硬性过滤的候选机台按参考收尾时间形成同班次窗口：

1. 取候选机台最早参考收尾时间 `referenceTime` 为基准 `t0`；
2. 定位 `t0` 所命中的班次为基准班次（半开区间 `[班次开始, 班次结束)`）；
3. 仅保留参考收尾时间落在基准班次区间 `[班次开始, 班次结束)` 内的候选机台；
4. 后续匹配项只能在该同班次窗口内比较；
5. 窗口外机台即使胎胚、规格、模壳或英寸更匹配，也不得越过窗口内机台。
6. 基准 MUST 取参考收尾时间 `referenceTime`，MUST NOT 取可开产时间 `switchStartTime + 换模总时长`；可开产时间会因晚班不能换模推迟到次日、再叠加换模总时长（默认 8h）而落到与收尾时间不同的班次，若用其定基准再按收尾时间过滤，会把包括基准机台在内的全部候选过滤为 0，导致新增排产 SKU 数为 0。

#### Scenario: 跨班次机台不得越过同班次窗口内机台

- **GIVEN** SKU 存在两台候选机台
- **AND** 机台 A 的参考收尾时间为 `t0`，落在班次 S 内
- **AND** 机台 B 的参考收尾时间晚于班次 S 结束时间
- **AND** 机台 B 的规格、模壳和英寸均优于机台 A
- **WHEN** 系统执行新增排产选机排序
- **THEN** 系统 MUST 只在基准班次 S 的 `[开始, 结束)` 窗口内选择机台
- **AND** 机台 B 不得参与本轮窗口内排序

### Requirement: 窗口内按单控、胎胚、模壳、规格、胶囊、英寸和机台编码逐层排序

系统 MUST 在同班次窗口内按以下顺序排序候选机台：

1. 单控拆分：试制 `0`，量试 `1`，小批量 `2`，普通机台 `3`，正规 SKU 使用单控整机候选 `4`；
2. 同胎胚：机台在产物料与 SKU 同胎胚为 `0`，否则为 `1`；
3. 同模壳：`isMouldSetPriorityMatched` 为 true 时为 `0`，否则为 `1`；
4. 同规格：机台前规格与 SKU 规格一致为 `0`，否则为 `1`；
5. 胶囊共用性：胶囊分组一致为 `0`，否则为 `1`；
6. 同英寸：英寸完全一致为 `0`，否则为 `1`；
7. 相近英寸：英寸差值按升序排序；
8. 机台编码：`machineCode` 作为最终兜底排序。

除上述层级外，定点机台、当天空闲、续作释放降级、其他 SKU 占用、最早可开产班次、连续可生产班次、可用总产能、尾部零散产能、普通机台优先、特殊支持能力数量、收尾时间、胎胚共用数量和机台顺序等既有比较方法 MAY 保留实现和诊断字段，但 MUST NOT 在新增排产选机台排序 Comparator 中调用。

#### Scenario: 单控拆分优先于同胎胚

- **GIVEN** 两台候选机台均在同班次窗口内
- **AND** 机台 A 为量试 SKU 可用单控拆分机台，但前物料与当前 SKU 不同胎胚
- **AND** 机台 B 为普通机台，前物料与当前 SKU 同胎胚
- **WHEN** 系统执行窗口内排序
- **THEN** 机台 A MUST 排在机台 B 之前

#### Scenario: 同胎胚优先于同规格

- **GIVEN** 两台候选机台均在同班次窗口内
- **AND** 机台 A 的前物料与当前 SKU 同胎胚但不同规格
- **AND** 机台 B 的前物料与当前 SKU 同规格但不同胎胚
- **WHEN** 系统执行窗口内排序
- **THEN** 机台 A MUST 排在机台 B 之前

#### Scenario: 同模壳优先于同规格

- **GIVEN** 两台候选机台均在同班次窗口内
- **AND** 两台候选机台与当前 SKU 的同胎胚判断结果一致
- **AND** 机台 A 与当前 SKU 同模壳但不同规格
- **AND** 机台 B 与当前 SKU 同规格但模壳不匹配
- **WHEN** 系统执行窗口内排序
- **THEN** 机台 A MUST 排在机台 B 之前

#### Scenario: 同模壳优先于胶囊共用

- **GIVEN** 两台候选机台均在同班次窗口内
- **AND** 机台 A 与当前 SKU 同模壳但胶囊不共用
- **AND** 机台 B 与当前 SKU 模壳不匹配但胶囊共用
- **WHEN** 系统执行窗口内排序
- **THEN** 机台 A MUST 排在机台 B 之前

### Requirement: 机台模套型号必须作为硬过滤淘汰模壳不匹配机台

系统 MUST 在新增排产选机候选机台过滤阶段，将机台模套型号与 SKU 普通模具模壳型号的匹配作为硬过滤：

1. 机台模套型号为空或包含"通用"时，视为模壳通配，适配所有 SKU；
2. 机台模套型号支持英文逗号分隔的多个值，SKU 普通模具模壳命中任一即通过；
3. SKU 普通模具模壳型号（`MdmModelInfo.shellStandard`）未命中机台模套型号时，MUST 淘汰该机台并记录 `MOULD_SET_MISMATCH`；
4. SKU 仅有模具到货关系（模具可用日期有值且无普通模具模壳可判）时，不因模壳不匹配淘汰该机台；
5. 该硬过滤 MUST 在寸口范围校验之后、特殊材料支持能力校验之前执行。

#### Scenario: 普通模具模壳未命中机台模套必须淘汰

- **GIVEN** SKU 普通模具的模壳型号为 `H380`
- **AND** 候选机台模套型号为 `H310,H338`
- **WHEN** 系统执行候选机台硬过滤
- **THEN** 该机台 MUST 被淘汰，记录原因为模套型号不匹配

#### Scenario: 多模套命中其一即通过硬过滤

- **GIVEN** SKU 普通模具的模壳型号为 `H450`
- **AND** 候选机台模套型号为 `H420,H450`
- **WHEN** 系统执行候选机台硬过滤
- **THEN** 该机台 MUST 通过硬过滤

#### Scenario: 仅到货模具不因模壳不匹配淘汰

- **GIVEN** SKU 模具关系仅有模具到货关系（模具可用日期有值），无普通模具模壳
- **AND** 候选机台模套型号未命中任何已知模壳
- **WHEN** 系统执行候选机台硬过滤
- **THEN** 该机台 MUST NOT 因模套型号不匹配被淘汰

#### Scenario: 空模套或通用模套通过硬过滤

- **GIVEN** 候选机台模套型号为空或包含"通用"
- **WHEN** 系统执行候选机台硬过滤
- **THEN** 该机台 MUST 通过硬过滤

### Requirement: 同模壳判断必须支持多模套、空模套通配、模具到货不降级

系统 MUST 按 SKU 模具的模壳型号与机台模套型号判断同模壳优先级：

1. 机台模套型号支持英文逗号分隔的多个值；
2. 机台模套型号为空时，视为模壳通配；
3. 机台模套型号包含“通用”时，视为模壳通配；
4. SKU 与模具关系中的模具可用日期有值时，该模具视为模具到货关系，不参与同模壳降级；
5. 普通模具必须使用 `MdmModelInfo.shellStandard` 命中机台模套型号后，才视为同模壳。

#### Scenario: 机台多模套命中 SKU 模壳

- **GIVEN** SKU 普通模具的模壳型号为 `H450`
- **AND** 候选机台模套型号为 `H420,H450`
- **WHEN** 系统判断同模壳优先级
- **THEN** 该机台 MUST 视为同模壳

#### Scenario: 模具到货关系不因模壳不匹配降级

- **GIVEN** SKU 的模具关系中模具可用日期有值
- **AND** 候选机台模套型号未命中该模具的模壳型号
- **WHEN** 系统判断同模壳优先级
- **THEN** 该机台 MUST 视为模壳中性，不得因模壳不匹配降低优先级

#### Scenario: 空模套或通用模套适配所有 SKU

- **GIVEN** 候选机台模套型号为空或包含“通用”
- **WHEN** 系统判断同模壳优先级
- **THEN** 该机台 MUST 视为同模壳

### Requirement: 过程日志必须可对账选机窗口和排序命中层级

系统 MUST 在新增排产选机台过程日志中输出可对账信息，至少包含：

1. 同班次基准班次序号；
2. 同班次窗口起点（班次开始时间）；
3. 同班次窗口截止（班次结束时间）；
4. 窗口筛选前候选数；
5. 窗口外过滤数；
6. 单台候选是否入收尾窗口；
6. 单台候选单控拆分、同胎胚、同模壳、同规格、胶囊共用、同英寸、相近英寸和机台编码等命中信息；
7. 排序层级名称 MUST 只体现单控拆分、同胎胚、同模壳、同规格、胶囊共用、同英寸、相近英寸和机台编码；
8. 收尾窗口 MUST 作为筛选窗口信息输出，不得作为窗口内 SortKey 层级输出。

#### Scenario: 日志可定位窗口外过滤原因

- **GIVEN** SKU 存在窗口内和窗口外候选机台
- **WHEN** 系统输出机台候选排序日志
- **THEN** 日志 MUST 展示同班次窗口起止时间和窗口外过滤数
- **AND** 候选明细 MUST 展示是否入收尾窗口

### Requirement: 正规 SKU 单控候选必须按整机释放时间参与窗口筛选

系统 MUST 在新增排产选机台时按单控动态粒度处理候选窗口：

1. 试制、量试、小批量 SKU 的单控候选按单边机台计算参考收尾时间；
2. 正规 SKU 的单控候选必须先收敛成 L/R 整机候选；
3. 正规 SKU 单控整机候选参考收尾时间 MUST 取 L/R 两侧释放时间的较晚值；
4. 缺少配对侧或任一侧未通过硬性约束时，不得把单边机台放入正规 SKU 候选窗口。

#### Scenario: 正规 SKU 单控窗口按两侧较晚释放时间判断

- **GIVEN** 正规 SKU 存在 `K1501L` 和 `K1501R` 两个单控侧候选
- **AND** `K1501L` 早于 `K1501R` 释放
- **WHEN** 系统执行新增排产选机窗口筛选
- **THEN** `K1501` 整机候选的参考收尾时间 MUST 等于 `K1501R` 释放时间

### Requirement: 强制重排时不续作的在机SKU不得占用当前排程窗口

系统 MUST 在强制重排完成续作SKU匹配后，按本次实际排程需求确定在机SKU的机台收尾时间：

1. 在机SKU本次仍需续作排程时，机台 MUST 保留该SKU的真实预计收尾时间；
2. 在机SKU本次无需排程、硫化余量为0或已列入未排时，机台预计收尾时间 MUST 取当前排程窗口首班开始时间；
3. 机台初始快照和强制重排继承结果的结束时间 MUST 同步更新，后续续作状态恢复、新增选机和换模计算不得重新读取目标日前一日的旧收尾时间；
4. 强制重排继承结果 MUST 保留用于识别机台当前物料和前规格，不得通过删除继承结果规避旧收尾时间。

#### Scenario: K1116在机SKU本次零余量未排

- **GIVEN** 强制重排窗口首班开始时间为 `2026-07-11 06:00:00`
- **AND** `K1116` 在机SKU为 `3302001311`，目标日前一日继承收尾时间为 `2026-07-11 22:00:00`
- **AND** `3302001311` 本次硫化余量为0并列入未排
- **WHEN** 系统完成续作与新增SKU分类
- **THEN** `K1116` 的机台运行态、初始快照和继承结果结束时间 MUST 同步为 `2026-07-11 06:00:00`
- **AND** 后续新增SKU换模不得再受 `3302001311` 的旧收尾时间限制

#### Scenario: 在机SKU本次仍需续作

- **GIVEN** 在机SKU已匹配为本次续作SKU
- **WHEN** 系统完成续作与新增SKU分类
- **THEN** 该续作机台 MUST 保留原预计收尾时间
- **AND** 系统 MUST NOT 将该机台提前释放到窗口首班
