## Purpose

规范新增排产阶段 SKU 选择候选硫化机台时的排序规则，确保系统先锁定最早收尾时间后 20 分钟内的机台池，再在该池内按胎胚、规格、模壳、胶囊和英寸逐层选择最优机台。

## Requirements

### Requirement: 新增排产选机台必须先筛最早收尾 20 分钟窗口

系统 MUST 在新增排产 SKU 选择候选机台时，先对通过硬性过滤的候选机台按参考收尾时间形成窗口：

1. 取候选机台最早参考收尾时间为 `t0`；
2. 仅保留参考收尾时间 `>= t0` 且 `<= t0 + 20分钟` 的候选机台；
3. 后续匹配项只能在该窗口内比较；
4. 窗口外机台即使胎胚、规格、模壳或英寸更匹配，也不得越过窗口内机台。

#### Scenario: 窗口外机台不得越过窗口内机台

- **GIVEN** SKU 存在两台候选机台
- **AND** 机台 A 的参考收尾时间为 `t0`
- **AND** 机台 B 的参考收尾时间晚于 `t0 + 20分钟`
- **AND** 机台 B 的规格、模壳和英寸均优于机台 A
- **WHEN** 系统执行新增排产选机排序
- **THEN** 系统 MUST 只在 `t0` 到 `t0 + 20分钟` 窗口内选择机台
- **AND** 机台 B 不得参与本轮窗口内排序

### Requirement: 窗口内按胎胚、规格、模壳、胶囊、英寸逐层排序

系统 MUST 在最早收尾 20 分钟窗口内按以下顺序排序候选机台：

1. 同胎胚；
2. 同规格；
3. 同模壳；
4. 胶囊共用性；
5. 同英寸；
6. 相近英寸；
7. 既有兜底排序。

既有兜底排序 MAY 继续包含定点机台、单控偏好、当天空闲、续作释放降级、产能画像、普通机台优先、特殊支持能力数量、机台顺序和机台编码等既有层级，但这些层级不得前置到胎胚、规格、模壳、胶囊和英寸之前。

#### Scenario: 同胎胚优先于同规格

- **GIVEN** 两台候选机台均在最早收尾 20 分钟窗口内
- **AND** 机台 A 的前物料与当前 SKU 同胎胚但不同规格
- **AND** 机台 B 的前物料与当前 SKU 同规格但不同胎胚
- **WHEN** 系统执行窗口内排序
- **THEN** 机台 A MUST 排在机台 B 之前

#### Scenario: 同模壳优先于胶囊共用

- **GIVEN** 两台候选机台均在最早收尾 20 分钟窗口内
- **AND** 机台 A 与当前 SKU 同模壳但胶囊不共用
- **AND** 机台 B 与当前 SKU 模壳不匹配但胶囊共用
- **WHEN** 系统执行窗口内排序
- **THEN** 机台 A MUST 排在机台 B 之前

### Requirement: 同模壳判断必须支持多模套、空模套通配、模具到货不降级

系统 MUST 按 SKU 模具的模壳型号与机台模套型号判断同模壳优先级：

1. 机台模套型号支持英文逗号分隔的多个值；
2. 机台模套型号为空时，视为模壳通配；
3. 机台模套型号包含“通用”时，视为模壳通配；
4. SKU 与模具关系中的模具可用日期有值时，该模具视为模具到货关系，不参与同模壳降级；
5. 普通模具必须使用 `MdmModelInfo.shellStandard` 命中机台模套型号后，才视为同模壳。

#### Scenario: 机台多模套命中 SKU 模壳

- **GIVEN** SKU 普通模具的模壳型号为 `H450`
- **AND** 候选机台模套型号为 `H420,H450`
- **WHEN** 系统判断同模壳优先级
- **THEN** 该机台 MUST 视为同模壳

#### Scenario: 模具到货关系不因模壳不匹配降级

- **GIVEN** SKU 的模具关系中模具可用日期有值
- **AND** 候选机台模套型号未命中该模具的模壳型号
- **WHEN** 系统判断同模壳优先级
- **THEN** 该机台 MUST 视为模壳中性，不得因模壳不匹配降低优先级

#### Scenario: 空模套或通用模套适配所有 SKU

- **GIVEN** 候选机台模套型号为空或包含“通用”
- **WHEN** 系统判断同模壳优先级
- **THEN** 该机台 MUST 视为同模壳

### Requirement: 过程日志必须可对账选机窗口和排序命中层级

系统 MUST 在新增排产选机台过程日志中输出可对账信息，至少包含：

1. 最早收尾窗口起点；
2. 最早收尾窗口截止；
3. 窗口筛选前候选数；
4. 窗口外过滤数；
5. 单台候选是否入收尾窗口；
6. 单台候选同胎胚、同规格、同模壳、胶囊共用、同英寸、相近英寸等命中信息；
7. 排序层级名称必须体现收尾窗口、同胎胚、同规格、同模壳、胶囊、同英寸和相近英寸。

#### Scenario: 日志可定位窗口外过滤原因

- **GIVEN** SKU 存在窗口内和窗口外候选机台
- **WHEN** 系统输出机台候选排序日志
- **THEN** 日志 MUST 展示窗口起止时间和窗口外过滤数
- **AND** 候选明细 MUST 展示是否入收尾窗口
