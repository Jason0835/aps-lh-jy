# 修复新增排产单机可满足时误拆量加机台

## 背景

2026-06-02 硫化排程中，多个新增排产 SKU 在一台机台已可满足日标准产能或加机台规则时，仍被拆到第二台机台，导致首台班次未排满或换活字块后重复新增机台。

典型表现：

- 物料 3302002661 在 K1111 已可覆盖窗口 8 班产能时，只排 64，C4-C8 未按 4/16/16/16/16/16 落满。
- 物料 3302001555 在 K1113 已可完成 108 硫化余量时，仍追加 K1111。
- 物料 3302001513 换活字块已形成有效硫化机台后，仍进入新增排产加机台。

## 目标

修复新增排产在日标准产能、窗口计划、换活字块承接都已满足时的误拆量和误加机台问题，保持欠产超过阈值时的动态增机台规则不变。

## 范围

- 新增排产 dayN 产能模拟后的机台数判断。
- 新增排产单机计划量分配。
- 换活字块/同物料已排机台对新增排产加机台判断的影响。
- `add-machine-rule` 主规格同步。

## 非目标

- 不调整候选机台排序、模具筛选、定点机台、特殊材料、工作日历等既有规则。
- 不新增表结构、配置项或独立排程流程。
