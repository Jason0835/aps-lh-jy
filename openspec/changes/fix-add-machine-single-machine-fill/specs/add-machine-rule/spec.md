## ADDED Requirements

### Requirement: 单机窗口理论产能满足时不得误加机台

系统必须（MUST）在本月前日累计欠产未超过阈值时，优先判断当前启用机台的 8 班窗口理论产能是否已覆盖 T～T+2 剩余窗口日计划；如果已覆盖，不得因为 T+1 或 T+2 单日 3 班理论产能小于当日计划而提前增加机台，也不得把当前机台计划量按多机台均分拆薄。

#### Scenario: 新增排产单机覆盖窗口计划

- **WHEN** 新增排产 SKU 的 `班产 * 8班` 已覆盖 T～T+2 窗口日计划总量
- **THEN** dayN 产能模拟必须返回当前机台数已满足
- **AND** 当前机台必须按真实可排窗口排产
- **AND** 系统不得生成第二台浅排机台或剩余未排结果

#### Scenario: 结构收尾误入严格策略但单机已覆盖

- **WHEN** SKU 被结构收尾或严格目标策略识别，但本月前日累计欠产未超过阈值，且当前一台机台 8 班理论产能已覆盖窗口日计划总量
- **THEN** 系统必须保持单机排产，不得按两台或多台均分目标量
- **AND** C7、C8 等后续可用班次不得因均分目标量被提前空置
