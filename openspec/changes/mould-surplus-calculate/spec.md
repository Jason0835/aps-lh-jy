调整硫化余量计算规则：

硫化余量 = MAX(月计划总量 - 已完成量 + 上月超欠产量, 0)

已完成量 = 月累计完成量（截至 T-1 日） + T 日排程晚班完成量。

上月超欠产量取值规则：
- 当月计划 lastMonthValidFlag = 1 时，取 lastMonthOverdueQty；
- 当 lastMonthValidFlag = 0、为空或无效时，按 0 处理。

要求：
1. T 日排程晚班完成量必须计入已完成量后再计算硫化余量；
2. 上月超欠产量只有有效标志为 1 时才参与计算；
3. 计算结果小于 0 时，硫化余量按 0；
4. 所有使用硫化余量的逻辑，包括收尾判断、排产目标量、共用胎胚余量为 0 不排等，必须统一使用修正后的硫化余量；
5. 尽量复用现有计算方法，必要时抽取公共方法，避免多处口径不一致。
