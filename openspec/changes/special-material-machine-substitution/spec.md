# 特殊材料硫化机置换需求变更文档

## Purpose

在硫化排程续作、换活字块、新增排产全部完成后，对仍未排上机台的特殊材料 SKU 执行兜底置换逻辑。从已排硫化排程结果中选择候选机台进行置换，确保特殊材料 SKU 尽量排上机台。

## Requirements

### Requirement: 置换触发时机必须在新增排产完成后

系统 MUST 在 S4.4 续作排产、S4.5 新增排产全部完成后，且 S4.6 结果校验保存之前，执行特殊材料硫化机置换逻辑。置换不参与前置抢机台，仅作为排程末尾的兜底补偿。

#### Scenario: 续作和新增排产完成后触发置换

- **WHEN** S4.5 新增规格排产完成，已形成初版硫化排程结果列表
- **THEN** 系统在 S4.5.1 步骤扫描未排结果中的特殊材料 SKU
- **AND** 若特殊材料 SKU 仍存在需排量且未匹配到任何机台，则触发硫化机置换逻辑

### Requirement: 置换目标必须排除已排上的特殊材料 SKU

系统 MUST 排除已在续作、换活字块、新增排产中排上任意机台的特殊材料 SKU，避免重复加机台或重复抢占机台。

#### Scenario: 特殊材料 SKU 已排上机台

- **WHEN** 特殊材料 SKU 已在排程结果列表中匹配到至少一台机台
- **THEN** 系统不对该 SKU 触发置换逻辑

#### Scenario: 特殊材料 SKU 仍有需排量且未排上

- **WHEN** 特殊材料 SKU 存在硫化余量或本次窗口需排量
- **AND** 该 SKU 在已排硫化排程结果中未匹配到任何机台
- **THEN** 系统对该 SKU 触发置换逻辑

### Requirement: 候选机台必须排除特殊材料 SKU 的机台

系统 MUST 在候选被置换机台中排除特殊材料清单内的 SKU，避免特殊材料之间互相挤占。

#### Scenario: 候选机台命中特殊材料 SKU

- **WHEN** 候选被置换机台上的 SKU 命中特殊材料清单
- **THEN** 系统将该机台排除出候选列表

### Requirement: 候选机台优先选择非续作 SKU 的机台

系统 MUST 优先从非续作 SKU 已排机台中选择候选被置换机台。非续作包括新增排产 SKU、换活字块后排产 SKU 和其他非续作来源的已排 SKU。只有当非续作机台中没有可置换候选时，才允许从续作 SKU 机台中选择。

#### Scenario: 非续作机台有候选

- **WHEN** 非续作 SKU 已排机台中存在硬匹配特殊材料 SKU 的候选
- **THEN** 系统仅从非续作候选中选择被置换机台

#### Scenario: 非续作机台无候选时回退续作

- **WHEN** 非续作 SKU 已排机台中不存在可置换候选
- **AND** 续作 SKU 机台中存在硬匹配候选
- **THEN** 系统从续作候选中选择被置换机台

### Requirement: 被置换机台按4级优先级逐层选择

系统 MUST 按以下优先级逐层匹配被置换机台，命中上层则不再比较下层：

1. 3天内喷砂清洗计划，喷砂时间越近越优先
2. 2天内月计划降模需求，降模时间越近越优先
3. 30天内精度保养计划，精度计划时间越近越优先
4. 胎胚库存低，库存越低越优先

#### Scenario: 命中3天内喷砂清洗

- **WHEN** 候选机台在排程日期起3天内存在喷砂清洗计划
- **THEN** 系统优先选择该机台作为被置换机台
- **AND** 多个机台命中时选择喷砂清洗计划时间最近的

#### Scenario: 未命中喷砂清洗但命中2天内月计划降模

- **WHEN** 候选机台无3天内喷砂清洗计划
- **AND** 候选机台在2天内存在月计划降模需求
- **THEN** 系统选择该机台作为被置换机台

#### Scenario: 未命中前两级但命中30天内精度计划

- **WHEN** 候选机台无3天内喷砂清洗和2天内月计划降模
- **AND** 候选机台在30天内存在精度保养计划
- **THEN** 系统选择该机台作为被置换机台

#### Scenario: 前三级均未命中时按胎胚库存低选择

- **WHEN** 候选机台未命中喷砂清洗、月计划降模和精度计划
- **THEN** 系统按被置换 SKU 对应胎胚库存判断
- **AND** 胎胚库存越低越优先被置换

### Requirement: 特殊材料 SKU 置换后必须遵守现有排程规则

系统 MUST 在特殊材料 SKU 置换到目标机台后，按现有逻辑判断换模或换活字块，并遵守晚班禁换模、换模/换活字块次数上限、首检、模具数量、单控机台、停机/清洗/精度不可生产时间等所有约束。排产量仍按现有硫化余量、班产、日标准修正、收尾目标量、胎胚库存、主销/常规补满等规则计算。

#### Scenario: 置换后需换模

- **WHEN** 特殊材料 SKU 置换到目标机台后需换模
- **THEN** 系统执行现有换模逻辑
- **AND** 遵守晚班禁换模、换模次数上限、换模均衡等约束

#### Scenario: 置换后换模约束不满足

- **WHEN** 特殊材料 SKU 置换到目标机台后因换模约束无法上机
- **THEN** 系统记录未排原因（如模具不足、晚班禁止换模、换模次数超限等）

### Requirement: 被置换 SKU 必须下机并回滚状态

系统 MUST 将被置换 SKU 从对应机台下机，并同步回滚其已排量、余量、模具占用、交替计划和收尾状态。被置换 SKU 若仍有未排余量，须记录未排原因。

#### Scenario: 被置换 SKU 下机

- **WHEN** 特殊材料 SKU 成功置换到目标机台
- **THEN** 被置换 SKU 从该机台下机
- **AND** 系统回滚机台状态、模具资源、换模计数和首检计数
- **AND** 被置换 SKU 记录未排原因为"被特殊材料SKU置换下机"

### Requirement: 模具交替计划备注必须体现置换类型

系统 MUST 在特殊材料 SKU 置换成功后生成的模具交替计划备注中写明命中的置换类型和被置换机台编码。备注格式为：{置换类型前缀} {机台编码}，被置换SKU：{物料编码}。

#### Scenario: 喷砂清洗置换备注

- **WHEN** 特殊材料 SKU 因3天内喷砂清洗计划命中置换
- **THEN** 模具交替计划备注为"喷砂+置换 {机台编码}，被置换SKU：{物料编码}"

#### Scenario: 月计划降模置换备注

- **WHEN** 特殊材料 SKU 因2天内月计划降模命中置换
- **THEN** 模具交替计划备注为"月计划降模+置换 {机台编码}，被置换SKU：{物料编码}"

### Requirement: 置换机台数按现有加机台规则计算

系统 MUST 按现有加机台规则计算特殊材料 SKU 的应需机台数，需要几台就尝试置换几台。每台置换独立校验并执行换模/换活字块、首检、晚班禁换模等约束。

#### Scenario: 需置换2台但仅成功1台

- **WHEN** 按加机台规则需要2台
- **AND** 最终只有1台满足置换和上机约束
- **THEN** 系统只置换成功的1台
- **AND** 剩余1台记录未排原因"特殊材料SKU置换不足"

### Requirement: 置换失败必须输出明确未排原因

系统 MUST 在置换失败时记录明确未排原因，区分未匹配到可置换机台、模具不足、晚班禁止换模、换模次数超限、单控机台约束不满足等情况。

#### Scenario: 无可置换机台

- **WHEN** 没有任何可置换机台
- **THEN** 特殊材料 SKU 记录未排原因"特殊材料SKU未匹配到可置换机台"

#### Scenario: 置换后仍未能排上

- **WHEN** 存在候选机台但置换后特殊材料 SKU 仍未能排上机台
- **THEN** 特殊材料 SKU 记录未排原因"特殊材料SKU置换后仍未能排上机台"
