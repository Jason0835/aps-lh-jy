调整硫化排程“欠产未超过阈值”时的新增排产加机台判断逻辑。

适用范围：
- 新增排产
- 续作排产
- 本月前日累计欠产 <= 欠产阈值
- 其余原有加机台规则保持不变，仅新增前置判断条件

核心调整：
原有逐日后看逻辑前，需要先判断当前机台数是否已经满足“当前日月计划量”。

规则背景：
保留原有逐日后看逻辑：
1. T 判断 T+1：
   - 当前机台数可满足 T+1 日计划，则 T 不加机台；
   - 当前机台数无法满足 T+1 日计划，则 T 加机台。

2. T+1 判断 T+2：
   - 当前机台数可满足 T+2 日计划，则 T+1 不加机台；
   - 当前机台数无法满足 T+2 日计划，则 T+1 继续加机台。

3. T+2 判断 T+3：
   - 当前机台数可满足 T+3 日计划，则 T+2 不加机台；
   - 当前机台数无法满足 T+3 日计划，则 T+2 继续加机台。

本次新增前置判断：
在每一天进入“后看下一日计划”前，必须先判断当前机台数是否已经满足“当前日月计划量”。

具体规则：
- 若当前机台数可满足当前日月计划量：
   - 当前日不加机台；
   - 不再继续判断下一日计划量；
   - 直接跳过当前日加机台判断。

- 若当前机台数无法满足当前日月计划量：
   - 才进入原有逐日后看逻辑；
   - 使用当前机台数判断下一日计划量是否满足；
   - 若下一日也无法满足，则当前日需要加机台。

示例：
- T 日先判断是否满足 T 日月计划量；
   - 满足：T 不加机台，不再判断 T+1；
   - 不满足：继续判断是否满足 T+1，决定 T 是否加机台。

- T+1、T+2 同理递推。

补充案例：
- 日计划为 `T=8、T+1=92、T+2=46、T+3=46`，单台机台日产能为46；
- T 日当前机台可满足8，T日不加机台；
- T+1当前机台不能满足92，但可以满足T+2的46，T+1不加机台；
- T+2当前机台可满足46，T+2不加机台；
- 最终只保留1台机台，不得因T+1当天计划92单独触发增机。

增机生效日期：
- 逐日判断在哪个业务日决定增加机台，该新增机台就从该业务日首个班次起生效；
- 新增机台不得在生效日前提前换模，应在生效业务日内首个允许换模班次开始换模；
- 若生效业务日首个班次为晚班且不可换模，新增机台应顺延到该业务日早班换模；换模完成时间正好等于早班结束时间时，首检数量按既有规则归入该早班；
- 新增机台的正常生产计划量不得落在生效日期之前，也不得早于换模和首检完成后的首个可生产班次；
- 同一业务日增加多台时，这些机台使用同一生效日期；不同业务日分别增加机台时，按新增顺序保留各自生效日期。

要求：
- 原有加机台逻辑整体保持不变；
- 仅新增“先判断当前日计划，满足则不后看”的前置条件；
- 尽量复用现有产能判断方法；
- 新增清晰注释和关键日志；
- 日志需包含：SKU、当前日期、当前机台数、当前日产能、当前日计划量、是否满足当前日、是否进入后看、后看日期、后看日计划量、是否加机台。

收尾目标保护：
- 续作 SKU 已按硫化余量/胎胚库存建立严格收尾目标后，历史欠产和日计划账本只用于追溯，不得再次抬高严格目标。
- 续作排产量已达到严格收尾目标时，补偿量必须为 0，不得生成补偿 SKU 回流新增排产。
- 非收尾 SKU 仍沿用原欠产阈值和逐日后看规则，不受本规则影响。
