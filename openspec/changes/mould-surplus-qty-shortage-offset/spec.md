调整硫化排程“收尾小余量不排产”规则。

一、参数

新增/复用硫化参数：
- 参数含义：硫化余量收尾允许的欠产偏差值
- 默认值：2
- 未配置时按默认值 2 处理

二、适用范围

适用于：
- 续作排产
- 换活字块排产候选
- 新增排产

三、命中条件

只有同时满足以下条件，SKU 本次才不进行排产：

1. SKU 为收尾；
2. 硫化余量 <= “硫化余量收尾允许的欠产偏差值”；
3. 该 SKU 前日排程结果中，T+1 日夜班未排满。

四、T+1 日夜班是否排满判断

判断该 SKU 在前日排程结果中的 T+1 日夜班排产量是否满足班产：

- 如果 T+1 日夜班排产量 >= 班产，视为已排满；
- 如果 T+1 日夜班排产量 < 班产，视为未排满；
- 如果前日排程数据为空，也视为未排满。

五、命中后的处理

如果满足以上条件：

- 本次不进行排产；
- 列入未排结果；
- 未排原因记录：收尾余量小于等于允许欠产偏差值，且前日 T+1 夜班未排满，本次不排产。

六、续作排产额外处理

如果是续作 SKU 命中该规则：

- 原本占用的续作机台必须释放；
- 释放后的机台优先参与换活字块排产；
- 如果不满足换活字块条件，再依次参与换模新增排产；
- 释放后不得继续被原续作 SKU 占用。

七、未命中情况

以下情况均按现有规则正常排产：

- SKU 非收尾；
- 硫化余量 > 参数值；
- SKU 为收尾且硫化余量 <= 参数值，但前日 T+1 夜班已排满。

八、实现要求

- 参数读取复用项目已有硫化参数配置机制；
- 判断逻辑需同时接入续作排产和新增排产；
- 续作场景要注意机台释放后流转顺序：换活字块优先，其次换模新增；
- 不影响现有收尾目标量规则；
- 不影响非收尾 SKU；
- 关键判断、未排原因、机台释放、后续流转都需要输出日志；
- 代码中添加清晰注释，说明这是“收尾小余量 + 前日 T+1 夜班未排满”的特殊不排产规则。

九、规则优先级

- S4.4 换活字块候选和 S4.5 新增排产必须在生成结果前执行统一未排判断；
- 同一 SKU 同时命中收尾小余量规则和仅历史欠产规则时，收尾小余量规则优先；
- 按收尾小余量规则写入未排时，未排数量取当前硫化余量，不得再按历史欠产量重复写入未排。
