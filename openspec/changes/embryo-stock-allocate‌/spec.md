请调整共用胎胚的胎胚库存分配规则。

背景：
多个 SKU 共用同一个胎胚时，需要按各 SKU 的实际消耗能力分配该共用胎胚库存。

核心规则：
1. 每个 SKU 的分配权重按以下公式计算：

   weight = 日硫化量 * 台数 * 模数

2. 每个 SKU 的分配占比：

   ratio = 当前 SKU weight / sum(所有共用 SKU weight)

3. 当前 SKU 实际应分配胎胚库存：

   allocatedStock = 共用胎胚总库存 * ratio

4. 注意：
  - 模数必须参与 weight 计算；
  - allocatedStock 计算时不要再乘模数；
  - 不允许实现成：共用胎胚总库存 * 占比 * 模数；
  - 否则会导致模数被重复放大，分配总量可能超过共用胎胚总库存。

5. 尾差处理：
  - 分配库存需要转整数；
  - 前 N-1 个 SKU 按项目现有取整口径处理；
  - 最后一个 SKU 必须分配剩余全部库存：

    lastAllocatedStock = 共用胎胚总库存 - sum(前面 SKU 已分配库存)

  - 确保所有 SKU 分配库存汇总严格等于共用胎胚总库存。

6. 异常场景：
  - 共用胎胚总库存 <= 0 时，各 SKU 分配库存为 0；
  - totalWeight <= 0 时，不进行比例分配，避免除零，按项目现有异常/兜底逻辑处理，并输出日志；
  - 单个 SKU 独占时，直接分配全部共用胎胚库存。

实现要求：
1. 尽量复用现有共用胎胚库存分配逻辑，只替换占比计算口径。
2. 分配过程增加日志，至少输出：
  - 胎胚编码/胎胚描述；
  - 共用胎胚总库存；
  - 每个 SKU 的日硫化量、台数、模数、weight、ratio、allocatedStock；
  - 尾差处理前后的最终分配结果。
3. 增加必要注释，说明“模数只参与权重计算，实际库存分配不再重复乘模数”。
4. 增加单元测试，覆盖：
  - 单 SKU；
  - 多 SKU 共用胎胚；
  - 日硫化量不同；
  - 台数不同；
  - 模数不同；
  - 存在整数尾差；
  - 共用胎胚库存为 0；
  - totalWeight 为 0；
  - 最后一个 SKU 分配剩余库存。
