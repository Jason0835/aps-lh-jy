请补充硫化排程模具号处理逻辑：

1. 新增排产结果中的模具号字段，保存本次实际分配使用的具体模具号；双模机台保存 2 个，单模机台保存 1 个，多个模具号用英文逗号分隔。

2. 续作排产结果中的模具号字段，取值来源为硫化在机信息 `in_machine_mould_code`，多个模具号以英文逗号分隔。续作排产生成结果时，必须保存当前机台实际在机占用的模具号，而不是 SKU 对应的全部模具号。

3. 续作排产时，`in_machine_mould_code` 中的在机模具号必须加入“已使用模具列表”，作为已占用模具处理。后续新增排产、SKU 增机台、换模分配模具时，都要排除这些已占用模具，避免重复分配。

4. 新增排产换模时，需要释放前物料实际占用的模具号，并为新物料分配可用模具号。选择模具时，从 SKU 可用模具列表中排除已使用模具，并优先选择共用 SKU 数量少的模具。

5. 续作 SKU 收尾后，如果该机台后续发生换模，需要释放该机台原先占用的在机模具号；释放对象应为续作机台实际占用的模具号。

6. 换活字块场景不释放模具号、不重新分配模具号，继续沿用当前机台在机模具号，因为换活字块使用同一副模具，只更换花纹块。
