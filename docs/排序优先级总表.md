# 排序优先级总表

## 一、续作SKU排序优先级

| 层级 | 排序维度 | 方向 | 说明 |
|------|----------|------|------|
| L1 | 锁定上机日期 | `deliveryLocked=true` 优先 | 有发货要求的 SKU 排最前 |
| L2 | 延误天数 | `delayDays` 升序 | 负数（提前上机）→ 0 → 正数（延误），越负越优先，null 排最后 |
| L3 | 结构全收尾命中 | 命中优先 | 结构下所有 SKU 在 N 天内全部收尾 |
| L4 | 结构内有已知收尾天数 | 有天数优先 | 收尾天数可计算才进入下一步 |
| L5 | 结构最晚收尾日 | 越晚越优先 | 收尾日越大（`-days`）越小越优先 |
| L6 | 高优先级待排量 | 降序 | 供应链优先级 04 |
| L7 | 周期排产待排量 | 降序 | 供应链优先级 05 |
| L8 | 中优先级待排量 | 降序 | 供应链优先级 06 |
| L9 | 常规待排量 | 降序 | 供应链优先级 07 |
| L10 | 开产靠后分 | 升序 | 仅开产模式：雪地胎 +100000，不同英寸 +100000 |
| L11 | 物料编码 | 字典序 | 最后兜底 |

## 二、新增SKU排序优先级

### 2.1 分组层（先决条件）

| 分组 | 分值 | SKU 类型 | 判定条件 |
|------|------|----------|----------|
| 试制组 | 0 | 试制 | `constructionStage=01(TRIAL)` |
| 量试组 | 1 | 量试 | `constructionStage=02(MASS_TRIAL)` |
| 正规组 | 2 | 其他 | 非试制非量试 |

### 2.2 正规组内排序

| 层级 | 排序维度 | 方向 | 说明 |
|------|----------|------|------|
| L1 | 定点机台 | 有定点优先 | 有限制定点机台的 SKU 优先 |
| L2 | 锁定上机日期 | `deliveryLocked=true` 优先 | 同续作 L1 |
| L3 | 延误天数 | 升序 | 同续作 L2 |
| L4 | 结构全收尾命中 | 命中优先 | 同续作 L3 |
| L5 | 结构有已知收尾天数 | 有天数优先 | 同续作 L4 |
| L6 | 结构最晚收尾日 | 越晚越优先 | 同续作 L5 |
| L7 | 高优先级待排量 | 降序 | 同续作 L6 |
| L8 | 周期排产待排量 | 降序 | 同续作 L7 |
| L9 | 中优先级待排量 | 降序 | 同续作 L8 |
| L10 | 常规待排量 | 降序 | 同续作 L9 |
| L11 | 开产靠后分 | 升序 | 同续作 L10 |
| L12 | 物料编码 | 字典序 | 兜底 |

### 2.3 试制/量试组内排序

| 层级 | 排序维度 | 方向 | 说明 |
|------|----------|------|------|
| L1 | 延误天数 | 升序 | 同续作 L2 |
| L2 | 排产量 | 降序 | `-targetScheduleQty`，排量大的优先 |
| L3 | 物料编码 | 字典序 | 兜底 |

## 三、机台排序优先级

| 层级 | 排序维度 | 方向 | 说明 |
|------|----------|------|------|
| L1 | 定点机台 | 定点优先(0) | 限制定点=0，其他=1 |
| L2 | 单控拆分优先级 | 按 SKU 类型 | 试制 0 → 量试 1 → 小批量 2 → 正规 4，非单控=3 |
| L3 | 其他 SKU 占用 | 未占用优先 | 未被其他 SKU 排程占用=0，已占=1 |
| L4 | 最早可开产班次 | 越小越优先 | 最早能开始生产的班次索引 |
| L5 | 连续可生产班次数 | 越多越优先 | 从第一班开始连续可生产班次 |
| L6 | 窗口内总可用产能 | 越大越优先 | 总可生产班次 × 班产 |
| L7 | 尾部零散产能 | 非尾部优先 | 有效班次 ≤ 2 或首班索引靠后判定为尾部 |
| L8 | 普通机台优先级 | 普通优先(0) | 仅非特殊 SKU 生效 |
| L9 | 特殊支持能力数 | 越少越优先 | 仅非特殊 SKU：统计宽基+芯片胎能力数 |
| L10 | 收尾时间 | 越早越优先 | 容忍度内相等，否则按 `estimatedEndTime` |
| L11 | 规格完全一致 | 一致优先(0) | 与机台 `previousSpecCode` 一致 |
| L12 | 英寸完全一致 | 一致优先(0) | 与机台 `previousProSize` 英寸一致 |
| L13 | 英寸接近度 | 差值越小越优先 | `\|skuInch - machinePreviousInch\|` |
| L14 | 胶囊共用性 | 共用优先(0) | 同规格胶囊组或同英寸胶囊组 |
| L15 | 胎胚共用数量 | 越多越优先 | 机台当前胎胚在其他 SKU 中出现的次数 |
| L16 | 机台排序号 | 升序 | `machineOrder` |
| L17 | 机台编码 | 字典序 | 兜底 |

## 四、汇总说明

### 4.1 续作 vs 新增 SKU 排序差异

| 差异点 | 续作 | 新增（正规组） | 新增（试制/量试组） |
|--------|------|----------------|---------------------|
| 定点机台排序 | 无此维度 | L1 有定点优先 | 无此维度 |
| 结构全收尾 | 三层子排序 | 三层子排序 | 无此维度 |
| 供应链优先级 | 四层逐级比较 | 四层逐级比较 | 无此维度 |
| 开产靠后分 | L10 | L11 | 无此维度 |
| 排产量兜底 | 无 | 无 | L2 排产量降序 |

### 4.2 排序惩罚常量

| 常量名 | 值 | 说明 |
|--------|-----|------|
| `OPEN_PRODUCTION_WINTER_TIRE_PENALTY` | 100000 | 开产模式雪地胎靠后分 |
| `OPEN_PRODUCTION_DIFFERENT_INCH_PENALTY` | 100000 | 开产模式不同英寸靠后分 |
| `DEFAULT_STRUCTURE_ENDING_DAYS` | 5 | 结构全收尾判定天数默认值 |
| `MAX_SHIFT_SLOT_COUNT` | 9 | 最大班次槽位数 |

### 4.3 单控机台评分对照

| 常量名 | 值 | 适用场景 |
|--------|-----|----------|
| `SINGLE_CONTROL_TRIAL_SCORE` | 0 | 试制 SKU + 单控机台 |
| `SINGLE_CONTROL_MASS_TRIAL_SCORE` | 1 | 量试 SKU + 单控机台 |
| `SINGLE_CONTROL_SMALL_BATCH_SCORE` | 2 | 小批量 SKU + 单控机台 |
| `SINGLE_CONTROL_NORMAL_MACHINE_SCORE` | 3 | 任意非单控机台（默认值） |
| `SINGLE_CONTROL_FORMAL_SCORE` | 4 | 正规 SKU + 单控机台 |
