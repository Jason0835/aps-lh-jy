==================================================
一、排程窗口定义
==================================================

本次续作排程窗口固定为3天8个班次：

T日：
- 早班
- 中班

T+1日：
- 晚班
- 早班
- 中班

T+2日：
- 晚班
- 早班
- 中班

班次映射如下：

class1 = T日早班
class2 = T日中班
class3 = T+1日晚班
class4 = T+1日早班
class5 = T+1日中班
class6 = T+2日晚班
class7 = T+2日早班
class8 = T+2日中班

注意：
不要按自然日的“早、中、晚 / 早、中、晚 / 早、中”理解。
第3个班次是T+1日晚班，不是T日晚班。

==================================================
二、T日需排产量计算规则
==================================================

T日需排产量不能直接等于T日月计划量，需要扣除T日晚班完成量，并叠加T-1日前日欠产/超产量。

公式：

T日需排产量 = T日月计划量 - T日晚班完成量 + T-1日前日欠产/超产量

其中：

T-1日前日欠产/超产量 = T-1日实际排产计划量 - T-1日已完成量

含义：
- 欠产为正数：说明T-1日计划未完成，需要追补到T日；
- 超产为负数：说明T-1日超额完成，需要抵扣T日计划。

示例：
T日月计划量 = 270
T日晚班完成量 = 90
T-1日欠产/超产量 = 0

则：
T日需排产量 = 270 - 90 + 0 = 180

由于T日只有早班、中班，如果5台机台班产均为18：
5台 × 18 × 2班 = 180
正好完成T日剩余需排量。

注意：
这里扣除的是“T日晚班完成量”，不是T日整天完成量。

==================================================
三、T+1、T+2每日需排产量滚动规则
==================================================

T+1、T+2按前一天排后差额滚动计算。

day1需排量 = day1月计划量 - T日晚班完成量 + T-1日前日欠产/超产量

day1排后差额 = day1需排量 - day1实际排产量

day2需排量 = day2月计划量 + day1排后差额

day2排后差额 = day2需排量 - day2实际排产量

day3需排量 = day3月计划量 + day2排后差额

day3排后差额 = day3需排量 - day3实际排产量

排后差额含义：
- > 0：欠产，需要下一天补；
- = 0：刚好完成；
- < 0：超产，可抵扣下一天计划。

==================================================
四、往后追补天数N参数
==================================================

新增或复用硫化参数：往后追补天数N。

参数含义：
在续作多机台降模减机台时，判断某台机台是否可以下机，不能只判断当天剩余机台是否满足当天产能，还要判断剩余机台是否可以满足后续N天的产能需求。

参数要求：
- 支持配置；
- 默认建议为1；
- 最小值为0；
- 最大值不能超过当前排程窗口剩余天数；
- 当前窗口最多3天，所以N最大不能超过3天窗口范围；
- 如果T日判断下机，最多只能看到T+1、T+2；
- 如果T+1日判断下机，最多只能看到T+2；
- 如果T+2日判断下机，则没有后续可追补天数。

举例：
如果N=1：
下机时需要判断剩余机台是否满足当天 + 下一天产能。

如果N=2：
下机时需要判断剩余机台是否满足当天 + 后两天产能。

如果N=0：
只判断当天产能。

==================================================
五、降模减机台触发条件
==================================================

仅针对续作排产，不要影响新增排产逻辑。

当满足以下条件时，进入降模减机台判断：

1. 当前SKU是续作SKU；
2. 当前SKU存在多台在线续作机台；
3. 当前T~T+2窗口内存在月计划量；
4. 当前在线续作机台总产能大于当前及追补窗口需求；
5. 减少部分机台后，剩余机台仍能满足：
   - 当天需排量；
   - 以及往后追补N天范围内的累计需排量。

不能只看当天。
例如T日看似可以减机台，但减完后T+1或T+2产能不够，则不能下机，必须保留。

==================================================
六、降模减机台判断逻辑
==================================================

对每个续作SKU，按天模拟T、T+1、T+2：

1. 计算当天需排量：
   - T日：day1月计划量 - T日晚班完成量 + T-1欠产/超产量；
   - T+1日：day2月计划量 + day1排后差额；
   - T+2日：day3月计划量 + day2排后差额。

2. 计算当天可排班次：
   - T日：2班，早、中；
   - T+1日：3班，晚、早、中；
   - T+2日：3班，晚、早、中。

3. 计算当前在线机台如果全部保留的产能。

4. 判断是否产能过剩。

5. 如果产能过剩，尝试按下机优先级逐台下机。

6. 每尝试下机一台，都要校验：
   - 剩余机台是否满足当天需排量；
   - 剩余机台是否满足往后追补N天范围内的累计需排量；
   - 不能超过当前3天窗口。

7. 如果满足，则允许该机台下机；
   如果不满足，则该机台必须保留。

==================================================
七、保留机台 / 下机机台排序规则
==================================================

优先复用项目已有胶囊次数、机台排序逻辑。

保留机台优先级：

1. 胶囊使用次数多的优先保留；
2. 胶囊使用次数相同，机台编码小的优先保留。

下机机台优先级：

1. 胶囊使用次数少的优先下机；
2. 胶囊使用次数相同，机台编码大的优先下机。

如果当前项目无法获取胶囊使用次数，则降级使用项目已有续作机台排序规则，排序靠后的机台优先下机。

==================================================
八、下机机台排产规则
==================================================

保留机台：
- 优先排满当天可排班次；
- 后续日期继续参与续作排产。

下机机台：
- 只补足当前日期剩余需排量；
- 如果剩余需排量大于等于一个班产，则排满一个班；
- 如果剩余需排量小于一个班产，则只排剩余量；
- 一旦补足当前日期需排量，该机台后续班次不再排当前SKU；
- 后续释放给其他SKU走换模或新增排产。

示例：
T日需排量 = 154
5台续作机台，班产16
T日只有早、中2班

前4台排满：
4 × 16 × 2 = 128

剩余：
154 - 128 = 26

第5台：
早班 = 16
中班 = 10
后续班次为空

==================================================
九、分阶段降模减机台
==================================================

降模减机台不是只在T日一次性判断，而是按天模拟。

例如：
T日需要3台；
T+1日只需要2台；
T+2日仍只需要2台。

则：
- T日保留3台；
- 第3台只排T日早班、中班；
- 从T+1日晚班开始，第3台下机；
- T+1、T+2只保留2台继续排。

对应最新版Excel场景3。

==================================================
十、单机台非收尾续作规则
==================================================

如果是非收尾 + 单机台续作：
- 不执行降模减机台；
- 只要开始续作，就排满窗口内可排班次；
- 即使当天月计划量较小、产生超产，也不提前停机；
- 超产量通过排后差额滚动抵扣后续日计划。

对应最新版Excel场景4。

示例：
T日计划量只有8；
单机台班产16；
T日有早、中2班；
则实际应排：
早班16 + 中班16 = 32。

不能因为计划只有8就提前停机。

==================================================
十一、T日无计划但后续有计划
==================================================

如果day1 = 0，但day2或day3有计划量：

1. T日早班、中班不排；
2. 从窗口内第一个有计划量的日期/班次开始续作；
3. 如果多台续作机台中，一台机台已能满足后续计划和追补N天判断，则其他机台应下机释放。

对应最新版Excel场景5。

示例：
day1 = 0
day2 = 32
day3 = 46

则：
T日早、中不排；
从T+1日晚班开始续作。

==================================================
十二、当前窗口无计划时释放机台
==================================================

如果续作SKU在当前T~T+2窗口内：

day1 = 0
day2 = 0
day3 = 0

则：
1. 本次不排该续作SKU；
2. 不允许占住机台等待窗口外未来计划；
3. 原在线机台释放给其他符合条件的SKU；
4. 被释放机台可进入换模排产或新增排产逻辑；
5. 如果月计划中存在该SKU，但当前窗口无计划，可记录未排原因：
   “当前排程窗口内无日计划量，等待后续滚动窗口排产。”

对应最新版Excel场景6。
